# 🎬 Movie Recommendation System

🚀 A modern **Movie Recommender Web App** built using Machine Learning & Streamlit.  
Get instant movie suggestions with posters, ratings & trailers.

---

## 🔥 Live Demo
https://movie-recommendation-system-bydevmanish.streamlit.app/

---

## ✨ Features
- 🎯 Content-based recommendation
- 🖼️ Posters via TMDB API
- ⭐ Ratings, genres, runtime
- 🎬 Watch trailer button
- ⚡ Fast Streamlit UI
- ☁️ Deployed on Streamlit Cloud

---

## 🧠 Tech Stack
Python, Pandas, Streamlit, TMDB API, gdown
